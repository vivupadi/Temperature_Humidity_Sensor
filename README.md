# Temperature & Humidity (IoT) System

A continuous temperature and humidity sensor to monitor and observe the changes using Grafana Dashboard. 
Alerts can be added

## Tech Stack

#### Hardware

1. ESP32 Dev C1
2. DHT11 Temperature and Humidity Sensor
3. LED
4. 220 Ohm Resistor

#### Software

1. HiveMQ
2. InfluxDB
3. Telegraf
4. Grafana
